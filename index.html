Über mich

Mein Interesse galt schon immer sowohl der IT und Sowareentwicklung als auch dem Design. In der Schule war Mathe mein bestes Fach, Programmieren fiel mir leicht.

Nach einem Studium an der Kunsthochschule begann meine berufliche Reise als Designerin. Fasziniert vom Aufkommen des Internets in den 90er Jahren wechselte ich als Quereinsteigerin in den IT Bereich und arbeitete zunächst angestellt, später selbständig im Bereich Webgestaltung und -prorammierung. In den letzten Jahren verhalf ich als Grafikerin mit Schwerpunkt Print einer Non-Profit-Organisation dazu, auf Events und Messen professionell und überzeugend aufzutreten.

Durch eine Weiterbildung zum Full Stack Web Developer habe ich mein Know-How im Bereich JavaScript-Programmierung auf einen aktuellen Stand gebracht und freue mich darauf, als Webentwicklerin meine Kenntnisse in einem innovativen Team einzubringen.

Chat App

Objective

Entwicklung einer Chat App für Mobilgeräte mit React Native. Die App stellt angemeldeten Usern ein Chat Interface zur Verfügung und ermöglicht das Senden von Bildern und Teilen des Ortes.

Features

Die User werden anonym mit Firebase authentifiziert. Der Chat-Verlauf und die Bilder werden in einer Firestore Datenbank gespeichert. Der Chat-Verlauf wird zudem lokal auf dem Mobilgerät gespeichert und steht auch im Offline-Modus zur Verfügung. User können Bilder aus der Bibliothek wählen oder neue Fotografien erstellen und die Bilder im Chat posten. User können ihren aktuellen Ort teilen, der im Chat auf einer Karte angezeigt wird.

Verwendete Technologien

  • React Native
  • Expo
  • Gifted Chat
  • Android Studio zum Testen
  • Google Firebase/Firestore

×

Angular Movie App

Objective

Eine clientseitige Anwendung für die bestehende Movie REST API unter Einsatz von Angular entwickeln und dokumentieren.

Features

Auf der Willkommensseite haben User die Möglichkeit, sich zu registrieren oder einzuloggen. Nach erfolgtem Login können User in der Filmdatenbank browsen und jeweils durch Klick auf einen Genre, Director oder Synopsis Button weitere Informationen zu einem Film erhalten. User haben die Möglichkeit, Favoriten zu speichern. In einer Profilansicht können User ihre persönlichen Informationen ändern, die Liste ihrer Favoriten sehen und bei Bedarf ihren Account löschen.

Dokumentation wurde mit Hilfe von Typedoc erstellt.

Verwendete Technologien

  • Angular Material
